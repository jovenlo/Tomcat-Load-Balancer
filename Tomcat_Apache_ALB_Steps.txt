Tomcat + Apache + AWS ALB Deployment Steps

1. Created EC2 Servers

LinuxServer1
LinuxServer2

Change hostname:

sudo hostnamectl set-hostname LinuxServer1

sudo hostnamectl set-hostname LinuxServer2


2. Update Server

sudo su -
yum update -y


3. Install Java

yum install java-17-openjdk -y

java -version


4. Install Tomcat

cd /opt

wget https://dlcdn.apache.org/tomcat/tomcat-10/v10.1.42/bin/apache-tomcat-10.1.42.tar.gz

tar -xvzf apache-tomcat-10.1.42.tar.gz

mv apache-tomcat-10.1.42 tomcat


5. Start Tomcat

cd /opt/tomcat/bin

chmod +x *.sh

./startup.sh

Test:

curl localhost:8080


6. Install Apache HTTPD

yum install httpd -y


7. Apache Reverse Proxy

Edit:

vi /etc/httpd/conf/httpd.conf

Add:

ProxyPreserveHost On

ProxyPass / http://localhost:8080/

ProxyPassReverse / http://localhost:8080/


8. SELinux Permission

setsebool -P httpd_can_network_connect 1


9. Start Apache

apachectl configtest

systemctl start httpd

systemctl enable httpd

systemctl status httpd


10. Test Apache Reverse Proxy

curl localhost

Output:
Apache Tomcat Page


11. Target Group

Target Type: Instances

Protocol: HTTP

Port: 80

Health Check Path: /

Register:
LinuxServer1
LinuxServer2


12. Application Load Balancer

Type:
Application Load Balancer

Scheme:
Internet Facing

Listener:
HTTP 80

Forward:
Load-Target-Group


13. Health Check

LinuxServer1 Healthy

LinuxServer2 Healthy


14. Final Output

Browser:

http://ALB-DNS-NAME

Output:

Apache Tomcat Welcome Page
